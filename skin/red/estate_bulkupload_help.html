  • 1. Preparation

    Two files are required when doing a bulk upload:

    1. a CSV file that records all of the product details
    2. a ZIP file that contains all of the product images

    If either one of these files is missing, it will result in an unsuccessful upload.

    You can find a .zip file which contains 2 sample files for them.

    Step1:  Go to “Your Admin” page on www.socexchange.com.au
    and click “Step 4: Add/Edit – real estate”.

    Step 2: In the ‘Bulk Product Import’ section, you can download the sample

    Step 3: Create your own CSV file with your products based on SOC format.

    Step 4: Pack all the images as a ZIP file.

    Step 5: Upload the files to the system.

  • 2. ZIP file

    All product images should be formatted in either .png / .gif / .jpg and compressed into a zip file. An image can only be used once during each upload operation.

    Please note that the following situation is not allowed.

    (An image cannot be shared by more than one item unless you are using duplication.)

  • 3. CSV file – Field / Value Table

    Below is a table that summarizes all the fields and field types in a CSV file.
    Fields that are labelled with a ‘red asterisk’ are mandatory.
    Any incorrect inputs will result in an unsuccessful upload for that particular item.
    However, all other items will not be affected.

    Notes:
    1) The item name has to be unique within your website, otherwise, the operation will be not work.
    2) Try to avoid using symbols like .,() etc in the item name, only use letters and number, as the system will ignore any symbols in the item names.
    3) Some fields are not required to fill when you choose different type of sale of the real estate. For example,

      If the typeOfSale equals
    • For Sale, then you can leave the following fields blank - auction, priceMethod
    • For Rent or Shared, then you can leave the following fields blank - auction
    • POA or Expressions of Interest, then you can leave the following fields blank - auction, price, priceMethod, negotiable
    • Auction, then you can leave the following fields blank - price, priceMethod, negotiable




    You can compare this table with those fields on the page.


  • 4. Upload

    Select the desired ‘CSV file’ in the ‘Products Information’ field. Select the desired ‘ZIP file’ in the ‘Product Images’ field.

    Next click ‘Import Products’. Your item upload process has now started.


  • 5. Report

    When the upload is completed, a brief summary will be visible at the top of the ‘Specify Product’ page - STEP 4.

    The number of upload successes and failures will be reported and the system will generate a report for the errors.
